HTML Window Events

Events triggered for the window object (applies to the body tag):

Event Description
onafterprint
  • after printing the web page
  • IE before printing the web page
onbeforeprint
  • before printing the web page
onbeforeunload
  • Before the page unload
  • Page Close
  • tag anchor
onerror
  • When error occurs while loading the HTML Elements
  • Can be use with HTML Element
onhashchange
  • When the anchor part has changed
  • Change the anchor part using location.hash
onload
  • When HTML page is loaded
  • Use
    • Set/Check Cookies
    • Run the JS Code when page loads up
ononline
  • When user is online
onoffline
  • When user is offline
onpageshow
  • Call when page is shown
onpopstate
  • When State is poped
    • onpopstate: When back
    • pushstate: When go to next
    • replacestate: Replace the state
onresize
  • When size of the window is changed
onunload
  • Close
  • Refresh
