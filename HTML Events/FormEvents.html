Form Events

When property of form get changed

Event Description
onblur
  • user leaves the input field
onchange
  • When input got changed
onfocus
  • When elements get focus
oninput
  • When input get changed
onselect
  • When user select the text of text input field
oncontextmenu
  • Opens context menu
  • When user performs Right-click on that div
oninvalid
  • When input field is not valid
onsubmit
  • Form Submit
onreset
  • Form reset
onsearch
  • Input type: search & after press enter
