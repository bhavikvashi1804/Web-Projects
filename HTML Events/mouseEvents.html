HTML Mouse Events

Triggered when mouse property got changed

Event Description
onclick
  • On Mouse Click
ondblclick
  • On Mouse Double Click
onmousedown
  • Mouse Left button is pressed and not released
onmouseup
  • Mouse Left button is released
onmouseover
  • when mouse pointer moves over an element
onmouseout
  • when mouse pointer moves out of an element
onmousemove
  • When Mouse hover
